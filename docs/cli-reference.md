# CLI Reference

## Core Commands

| Command | Description |
|---------|-------------|
| `vaultbot run` | Start the bot |
| `vaultbot setup` | Run setup wizard |
| `vaultbot status` | Show system status |
| `vaultbot version` | Show version |

## Agent Commands

| Command | Description |
|---------|-------------|
| `vaultbot agent create <id>` | Create an agent |
| `vaultbot agent list` | List agents |
| `vaultbot agent delete <id>` | Delete an agent |
| `vaultbot agent bind <id> <channel>` | Bind agent to channel |

## Channel Commands

| Command | Description |
|---------|-------------|
| `vaultbot channel add <name>` | Add a channel |
| `vaultbot channel remove <name>` | Remove a channel |
| `vaultbot channel status` | Show channel status |

## Config Commands

| Command | Description |
|---------|-------------|
| `vaultbot config get <key>` | Get config value |
| `vaultbot config set <key> <value>` | Set config value |
| `vaultbot config list` | List config keys |

## Plugin Commands

| Command | Description |
|---------|-------------|
| `vaultbot plugin install <name>` | Install plugin |
| `vaultbot plugin list` | List plugins |
| `vaultbot plugin enable <name>` | Enable plugin |
| `vaultbot plugin disable <name>` | Disable plugin |
