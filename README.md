# ZenBot

A security-first, open-source AI agent bot for messaging platforms.

## Quick Start

```bash
pip install zenbot-agent
zenbot init
zenbot run
```

## Features

- **Security-first**: Encrypted credentials, mandatory auth, plugin sandboxing
- **Multi-platform**: Telegram, Discord, WhatsApp, Signal
- **Pluggable LLM**: Claude, OpenAI GPT, local models
- **Plugin system**: Sandboxed, signed, permission-controlled extensions
